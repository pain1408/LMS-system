Common Issues & Solutions

Login Problems

Can't access your account?

  • Check your Student ID and password
  • Clear browser cache and cookies
  • Try password reset
  • Contact support if issue persists

Book Pre-booking

Issues with reserving books?

  • Ensure book is available
  • Check your borrowing limit
  • Clear any outstanding fines
  • Try refreshing the page

Payment Issues

Problems paying fines?

  • Check mobile banking balance
  • Verify payment details
  • Upload clear transaction screenshot
  • Wait for processing (2-4 hours)

File Upload

Can't upload study notes?

  • Check file format (PDF only)
  • Ensure file size under 10MB
  • Try different browser
  • Check internet connection
